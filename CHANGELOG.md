# Changelog

All notable changes to this project will be documented in this file.

## [2.0.0] - 2026-01-06

### Added
- Complete documentation overhaul
- Project structure improvements
- Comprehensive README with table of contents
- Troubleshooting guide
- Prerequisites and installation instructions

### Changed
- Updated README.md with detailed usage instructions for all algorithms
- Improved project organization documentation

## [1.1.0] - 2019-05-16

### Added
- Invasive Weed Optimization (IWO) implementation
- New YPEA119 framework integration

### Changed
- Language modifications
- Updated README with IWO usage instructions

### Fixed
- Removed junk files from repository

## [1.0.0] - 2019-05-16

### Added
- Initial public release
- Genetic Algorithm implementation
- Artificial Bee Colony algorithm
- Bees Behavior Algorithm
- Simulated Annealing algorithm
- PSO example code
- Flight test data from TREX 550 helicopter
- 3D visualization and animation capabilities
- Parameter visualization script (PopulCheck.m)

### Documentation
- Basic README with algorithm descriptions
- IFAC paper documentation
- Algorithm-specific licenses

## [0.9.0] - 2017-03-06

### Added
- README documentation
- Project description and usage instructions

### Changed
- Initial commit with core functionality
- Flight data processing scripts
- State-space model implementation

## [0.1.0] - 2017-01-18

### Added
- Initial commit
- Basic project structure
- Helicopter state-space model based on Mettler's framework
- Initial genetic algorithm implementation
- Flight data collection utilities
