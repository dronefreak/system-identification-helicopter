## Adding Documentation

When adding new documentation:
1. Place papers and reports in this directory
2. Save figures in `figures/` subdirectory
3. Use descriptive filenames
4. Update this README with references
5. Include proper citations in CITATION.cff

## Documentation Standards

- Use Markdown (.md) for text documentation
- PDF for final reports and papers
- PNG/SVG for figures (avoid JPEG for technical diagrams)
- Include source files when possible (e.g., .tex for LaTeX)
